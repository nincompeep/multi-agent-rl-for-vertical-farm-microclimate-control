How It Works

Three independent agents learn to control their own shelf tier using continuous plant health signals

👁️

Observe

Each agent observes its own tier's temperature, CO₂, humidity, light intensity, and crop state every timestep.

→
🧠

Decide

The PPO neural network processes observations and outputs 6 continuous control actions — LED, heating, ventilation, and nutrient settings.

🌱

Reward

About This Project

Traditional vertical farms use static rule-based controllers that treat the entire farm as one uniform zone, ignoring microclimate variations across shelf tiers caused by heat, light, and airflow differences.

This project proposes a Multi-Agent Reinforcement Learning framework where each shelf tier is managed by an independent PPO agent. The key innovation is phenotyping-based reward shaping — using continuous, non-destructive plant metrics like Leaf Area Index and a chlorophyll-based stress indicator.

Phenotyping Reward Formula
Rt = (LAIt − LAIt−1) × 20 − Stemp + SCO₂ + Shum − |ūt| × 0.05
■ LAI growth reward   ■ Stress penalty   ■ Energy penalty
